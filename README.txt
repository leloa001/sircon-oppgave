server.js filen skal kjøres med node, har du ikke node kan du laste det ned her:

https://nodejs.org/en

for å initiere node i prosjektet må du åpne filplaseringen
i en terminal (gjerne VS code sin innebygde terminal)
og skrive inn følgende kommando:

npm init -y

når du har gjort dette kan du skrive inn neste kommando:

npm install socket.io

dette vil instalere socket.io biblioteket som jeg har brukt til websocket
delen av oppgaven

nå kan du åpne index.html filen eller hoste den på en server.
